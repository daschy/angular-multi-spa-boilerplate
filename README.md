## How add another SPA

1. Copy directory app/spa1

2. paste it in app/

3. rename app/spa1 --> app/fooApp

3. copy and rename spa1.html -> fooApp.html

3. in fooApp.html rename all the occurence "spa1/" -> "fooApp/"
